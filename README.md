# EdgeRusher

AI-powered NFL game analysis platform providing insights and competitive edge for football enthusiasts.

## Tech Stack
- Next.js 15 (App Router)
- Supabase (PostgreSQL + Auth)
- Stripe (Payments)
- OpenAI GPT-4o-mini
- Vercel (Hosting)

## Features
- Daily AI-generated game analysis
- Multi-AI Pro Chat (GPT-4o-mini, Claude, Gemini)
- Credit-based system
- Real-time NFL data from ESPN API
